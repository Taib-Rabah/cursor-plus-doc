## Welcome!

Welcome to the CursorPlus documentation. Here you'll find everything you need to know about the extension.

## What is CursorPlus?

CursorPlus is a VS Code extension that gives you more cursor commands to move faster and more freely in the editor. Currently it has over 60 commands grouped into 7 groups.

## Install

[Install CursorPlus from VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=TRDev20.cursor-plus).

Or you can follow these steps:

<Steps>

<Step>

Open VS Code

</Step>

<Step>

Press `Ctrl+Shift+P` to open the command palette

</Step>

<Step>

Type <CopyText text="Extensions: Install Extensions" code /> and press Enter

</Step>

<Step>

Search for <CopyText text="CursorPlus" />

</Step>

<Step>

Click the extension and install it

</Step>

</Steps>
