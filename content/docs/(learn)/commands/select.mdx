---
title: Select
description: Select text between cursor and target position
---

<Callout className="mt-0">

Similar to the move commands but to select from [Selection.anchor](/docs/info#selection-parts) to the target position.

</Callout>

## Available commands

| Name            | id                                                       | description                                        |
| --------------- | -------------------------------------------------------- | -------------------------------------------------- |
| Line Start      | <CopyText text="cursorPlus.select.lineStart" code />     | Select to start of line                            |
| Up Line Start   | <CopyText text="cursorPlus.select.upLineStart" code />   | └─ in previous line                                |
| Down Line Start | <CopyText text="cursorPlus.select.downLineStart" code /> | └─ in next line                                    |
| Text Start      | <CopyText text="cursorPlus.select.textStart" code />     | Select to first non-whitespace character           |
| Up Text Start   | <CopyText text="cursorPlus.select.upTextStart" code />   | └─ in previous line                                |
| Down Text Start | <CopyText text="cursorPlus.select.downTextStart" code /> | └─ in next line                                    |
| Text End        | <CopyText text="cursorPlus.select.textEnd" code />       | Select to last non-whitespace character            |
| Up Text End     | <CopyText text="cursorPlus.select.upTextEnd" code />     | └─ in previous line                                |
| Down Text End   | <CopyText text="cursorPlus.select.downTextEnd" code />   | └─ in next line                                    |
| Line End        | <CopyText text="cursorPlus.select.lineEnd" code />       | Select to end of line                              |
| Up Line End     | <CopyText text="cursorPlus.select.upLineEnd" code />     | └─ in previous line                                |
| Down Line End   | <CopyText text="cursorPlus.select.downLineEnd" code />   | └─ in next line                                    |
| Line            | <CopyText text="cursorPlus.select.line" code />          | Select entire line content                         |
| Up Line         | <CopyText text="cursorPlus.select.upLine" code />        | └─ previous line                                   |
| Down Line       | <CopyText text="cursorPlus.select.downLine" code />      | └─ next line                                       |
| Text            | <CopyText text="cursorPlus.select.text" code />          | Select from first non-whitespace to last character |
| Up Text         | <CopyText text="cursorPlus.select.upText" code />        | └─ in previous line                                |
| Down Text       | <CopyText text="cursorPlus.select.downText" code />      | └─ in next line                                    |
| Wrapper         | <CopyText text="cursorPlus.select.wrapper" code />       | Choose a select command from a quick-pick menu     |

<Callout title="Wrapper command?">

Learn more in the [Wrapper command](/docs/commands/wrapper) documentation.

</Callout>

## Args

<Callout>

This command takes no arguments.

</Callout>

## Examples

### Keyboard shortcut

```json title="keybindings.json"
[
  {
    "key": "ctrl+k ,",
    "command": "cursorPlus.select.line",
    "when": "editorTextFocus"
  }
]
```
